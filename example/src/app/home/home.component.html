NGX-PUZZLE

拖拽式仪表盘构建器

基于 Angular 的可视化拖拽组件库，像拼图一样快速构建响应式数据仪表盘。 支持图表、表格、文本和控件的自由组合。

📋
📝

核心特性

🎯

拖拽编辑器

直观的拖拽界面，支持网格吸附布局，轻松创建专业仪表盘

🧩

丰富组件

内置图表、表格、文本、控件等多种组件，满足各种数据展示需求

⚡

现代架构

基于 Angular Standalone + Signals，OnPush 优化性能

🔗

外部数据

灵活的数据绑定契约，轻松连接真实 API 或模拟数据

👀

预览保存

通过外部服务钩子实现预览和保存功能

📱

响应式

完全响应式设计，适配各种屏幕尺寸

组件预览

📊

图表组件

基于 ECharts，支持柱状图、折线图、饼图等多种图表类型

📋

表格组件

支持数据透视、分组聚合的高级表格组件

📝

文本组件

支持富文本编辑，标题、段落等多种文本样式

🎛️

控件组件

日期选择器、下拉框等交互控件，支持联动筛选

安装依赖

2

import { NgxPuzzleEditorComponent } from '@zhongmiao/ngx-puzzle';
3

<ngx-puzzle-editor></ngx-puzzle-editor>
